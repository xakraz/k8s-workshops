## 1 - Bootstrap

Clone the repo:
```
$ git clone https://github.schibsted.io/xavier-krantz/k8s-workshops
```

Get `Kubectl`:
```
$ cd k8s-workshops/1
$ ./scripts/get_kubectl.sh
```

And follow the instructions
